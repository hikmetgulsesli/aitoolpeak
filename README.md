# AIToolPeak

AI Tools review and comparison blog built with React 19 + Vite 6 + Express.

## Tech Stack
- React 19 + Vite 6
- Express.js backend
- TypeScript
- Tailwind CSS v4
- React Router DOM v7

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm start
```

## Environment
Copy `.env.example` to `.env` and configure:
- PORT: Server port (default: 3525)
- SITE_URL: Production site URL
