# Setting Up Ollama for Local AI: A Complete Guide

Running AI models locally with Ollama gives you complete privacy and control.

## Why Local AI?

- **Privacy**: Your code never leaves your machine
- **Cost**: No API fees
- **Control**: Run any model you want

## Installation

```bash
# macOS
brew install ollama

# Linux
curl -fsSL https://ollama.com/install.sh | sh
```

## Running Your First Model

```bash
ollama run codellama
```

## Best Models for Coding

| Model | Size | Best For |
|-------|------|----------|
| CodeLlama | 7B-70B | General coding |
| DeepSeek Coder | 6.7B-33B | Code completion |
| Mistral | 7B | Fast responses |

## Integration with Editors

Use the Continue.dev extension for VS Code integration.

Local AI is now viable for most development tasks.
