## Cost Optimization Strategies

### 1. Use Caching
Cache repeated prompts for 50% discount.

### 2. Choose Right Model
- Simple Q and A: Haiku or Mini
- Code completion: Claude Haiku
- Code review: Claude Sonnet
- Complex reasoning: GPT-4o or Opus

### 3. Batch Processing
Process multiple prompts at once for 50% discount.
